# Data Science Projects

This repository contains a collection of **Data Science and Basic Machine Learning projects** implemented in **Python**.  
The projects demonstrate end-to-end workflows including **data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation** using real-world datasets.

---

##  Objectives
- Build a strong foundation in **Data Science and Machine Learning**
- Apply theoretical concepts to **real-world datasets**
- Practice **data cleaning, EDA, and feature engineering**
- Develop and evaluate **machine learning models**
- Improve problem-solving and analytical skills

---

##  Tools & Technologies
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **Techniques:** EDA, Feature Engineering, Model Training & Evaluation
